Latest Projects

1 - Solar Tracking System

A solar tracking system is a device that orients solar panels or mirrors towards the sun throughout the day to maximize energy capture. By dynamically adjusting the position of the solar collectors, these systems can increase energy efficiency by 20-50% compared to fixed installations. Solar trackers can be single-axis, moving along one direction, or dual-axis, adjusting both horizontally and vertically. They are widely used in both photovoltaic and concentrated solar power systems.

2 - Tic-Tac-Toe

A Tic-Tac-Toe project involves creating a simple, two-player game where participants alternate marking X or O in a 3x3 grid, aiming to get three in a row horizontally, vertically, or diagonally. The project can be developed using various programming languages, such as Python or JavaScript, and serves as an excellent introduction to concepts like game logic, control flow, and user input handling. It typically includes features like a user-friendly interface, input validation, and a mechanism to detect winning or draw conditions.

3 - Three Layer Authentication

A three-layer authentication system is a security mechanism that uses multiple levels of verification to ensure the identity of a user. The first layer typically involves "something you know," like a password or PIN. The second layer incorporates "something you have," such as a smartphone or security token, often verified through a one-time code. The third layer is "something you are," which includes biometric verification like fingerprints, facial recognition, or voice authentication. This layered approach significantly enhances security by requiring different types of evidence to access sensitive data or systems
